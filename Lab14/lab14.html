¿Qué es ODBC y para qué es útil?

es un icono en el Panel de control de Microsoft® Windows®. Para el programador de aplicaciones, es una biblioteca que contiene rutinas de acceso a datos. Para muchos otros, es la respuesta a todos los problemas de acceso a bases de datos que se hayan imaginado.

¿Qué es SQL Injection?

SQL injection es un código de ataque que puede destruir la base de datos, es una de las técnicas de hackeo más comunes.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Utilizando usuarios personalizados con privilegios muy limitados, empleando sentencias preparadas con variables vinculadas que son proporcionadas por PDO, MySQLi y otras bibliotecas,comprobando si la entrada proporcionada tiene el tipo de datos previsto.
