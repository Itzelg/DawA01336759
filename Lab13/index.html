Manejo de sesiones con php


PHP

¿Por qué es importante hacer un session_unset() y luego un session_destroy()?

Para poder eliminar correctamente las variables globales y después destruir la sesión del usuario.

¿Cuál es la diferencia entre una variable de sesión y una cookie?

Una variable de sesión es una variable que se almacena mientras la sesión del usuario está abierta, mientras que una variable cookie se almacena en el disco duro del usuario por lo que permanece más tiempo que la variable de sesión.

¿Qué técnicas se utilizan en sitios como facebook para que el usuario no sobreescriba sus fotos en el sistema de archivos cuando sube una foto con el mismo nombre?

Facebook revisa las variables de sesión para ver si un archivo se repite o no.

¿Qué es CSRF y cómo puede prevenirse?

Es un ataque en el cual el atacante provoca que el usuario ejecute una acción de forma no intencionada en una aplicación en la que había iniciado sesión. Para prevenirlo se pueden utilizar tokens de sesión, frameworks con soporte ya instalado como Spring, o utilizando otros programas para proteger aplicaciones web.
