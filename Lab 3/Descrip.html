URL

Uniform Resource Locutor (URL), es una secuencia de caracteres que nos permite identificar recursos dentro del Internet para que puedan ser utilizados.

MIME

Multipurpose Internet Mail Extensions (MIME) es un tipo estandarizado que nos permite indicar el formato original de un documento.

Métodos y códigos de errores HTTP

Los códigos de errores y métodos HTTP, describen de forma abreviada la respuesta HTTP. Los errores, dependiendo del número, pueden ser ocasionados por diferentes factores:

  • 1XX Respuestas informativas
  • 2XX Peticiones correctas
  • 3XX Redirecciones
  • 4XX Errores del cliente
  • 5XX Errores de servidor

    XML es un subconjunto de SGML(Estándar Generalised Mark-up Language) simplificado y adaptado a Internet.

    XHTML

    XHTML (Lenguaje de Marcado de Hipertexto Extensible) es una versión mejorada de HTML, para remplazar a HTML ante su limitación de uso con las herramientas basadas en XML.

    JS

    JavaScript es el lenguaje de programación de HTML y la Web.

    PHP

    PHP (Hypertext Preprocessor) es un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML.

    JSP

    Java Server Pages (JSP) nos permite crear dinámicamente plataformas independientes a métodos basados en aplicaciones Web.

    Ajax

    Con Ajax nos es posible leer datos de un servidor web, actualizad una pagina web sin cargar la página o enviar datos a un servidor web.
