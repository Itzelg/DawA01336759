Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Tenemos que tener mucho cuidado en dónde la podemos utilizar, debido a que esta regla de estilo tomará más importancia que todas las demás reglas de estilo.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Primero debemos de tener cuidado con los pixeles de la imagen, para poderla visualizar en cualquier pantalla, también tenemos que cuidar el estilo de nuestra página para que todo sea visible dentro de ella.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Depende de lo que quieras hacer, por ejemplo pt es recomendable cuando lo que estas haciendo se va a imprimir, cuando utilices % es para que al ver la página se ajuste al tamaño de la pantalla y así con cada uno.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque hace que se cargue más rápido el archivo, ya que aunque es más complicado entenderlo para nosotros, la computadora la puede entender más rápido.
