¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications)?

Permite crear sitios rápidos, dinamicos y guardar recursos.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

Realizar una mala práctica con AJAX, puede afectar el control de acceso al sistema. AJAX agrega nuevos niveles de complejidad a la arquitectura del sistema. Todas las validaciones son realizadas del lado del cliente.

¿Qué es JSON?

JavaScript Object Notation: Es una sintaxis para almacenar e intercambiar datos en una aplicación.
