Ventajas y desventajas de su uso.

Ventajas Desventajas
Es el motor de base de datos objeto-relacional más utilizado a nivel mundial. El precio es muy elevado para una licencia de base de datos.
Permite el uso de particiones para la mejora de la eficiencia. El coste de la información es elevado y hay muy poca información sobre los asuntos tecnicos del gestor
Puede usarse en todas las plataformas y es la base de datos con más orientación hacia INTERNET. Existen muchas versiones que se realizaron hasta alcanzar la estabilidad en la version 8.0.3, lo que puede causar confusión.
