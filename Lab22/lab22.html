Integración con servicios web


¿Qué ventajas y desventajas tiene la integración de tus aplicaciones web con servicios web desarrollados por terceros?

Ventajas

Nos ayudan a desarrollar proyectos con mayor rapidez, ya que utilizamos elementos de la API, en lugar de tener que crearlos, también mejoran la imagen de nuestro sistema.

Desventajas

Las condiciones de la API pueden cambiar, los servicios pueden volverse de paga o por falta de mantenimiento quedar disfuncionales, estamos sujetos a la disponibilidad del servicio de estos, si se cae las funciones de nuestro sistema que utilizan la API quedan inhabilitadas.
